# BookApi - Node & MongoesDB

CRUD Books


## Deployment

First we need to install node-modules

```bash
  npm i node-modules
```

For the routing api we need express
```bash
 npm i express
```

To store the data we are using mongoose
```bash
  npm i mongoose
```
and create database BooksAPI and collection book
